# Sorteador Social MVP

Aplicación web simple para realizar sorteos de Instagram/TikTok/manuales.

## Features
- Sorteos aleatorios seguros
- Eliminación automática de duplicados
- Importación de TXT/CSV
- Historial local
- Exportación CSV
- Compartir resultados
- Persistencia en localStorage
- UI responsive

## Ejecutar

```bash
npm install
npm run dev
```

Abrir:

http://localhost:3000
